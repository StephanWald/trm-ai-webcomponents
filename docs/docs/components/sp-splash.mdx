# sp-splash

Full-screen modal overlay web component with backdrop blur, a gradient header, and smooth fade/slide animations. Content is placed using named slots: `logo`, `title`, `subtitle`, `body`, and `footer`. The splash can be dismissed via the close button, pressing Escape, or clicking the backdrop. Use `show()` and `hide()` methods for programmatic control, or set the `open` attribute directly.

:::info
`sp-splash` renders as a full-screen overlay that covers the entire viewport. The examples below use a button to trigger `show()` so you can see the component appear interactively.
:::

## Installation

Install from a [GitHub Release](https://github.com/StephanWald/trm-ai-webcomponents/releases):

```bash
npm install https://github.com/StephanWald/trm-ai-webcomponents/releases/download/v0.0.1/skillspilot-webcomponents-0.0.1.tgz
```

Or load via script tag:

```html
<script type="module" src="https://stephanwald.github.io/trm-ai-webcomponents/wc/skillspilot.esm.js"></script>
```

### Programmatic Control

Use `show()` and `hide()` methods to control the splash from external buttons or application logic.

<LiveExample height="420px" code={`
<div style="display:flex;gap:8px;">
  <button id="openBtn" style="padding:8px 16px;cursor:pointer;">Open</button>
  <button id="closeBtn" style="padding:8px 16px;cursor:pointer;">Close</button>
</div>

<sp-splash id="splash2">
  <span slot="title">Programmatic Control</span>
  <span slot="subtitle">Controlled via show() and hide() methods</span>
  <div slot="body" style="padding:16px 0;">
    <p>This splash was opened via the <code>show()</code> method and can be closed programmatically via <code>hide()</code> or using the dismiss controls.</p>
  </div>
</sp-splash>

<script>
  const splash = document.getElementById('splash2');
  document.getElementById('openBtn').addEventListener('click', function() { splash.show(); });
  document.getElementById('closeBtn').addEventListener('click', function() { splash.hide(); });
<\/script>
`} />

### Event Handling

Listen to the `splashClose` event to know when the splash is dismissed and why (`button`, `escape`, or `backdrop`).

<LiveExample height="420px" code={`
<button id="showBtn3" style="padding:8px 16px;cursor:pointer;">Show Splash</button>
<div id="closeLog" style="margin-top:12px;font-family:monospace;font-size:0.85rem;color:#555;"></div>

<sp-splash id="splash3">
  <span slot="title">Event Handling Demo</span>
  <span slot="subtitle">Dismiss via X button, Escape, or backdrop click</span>
  <div slot="body" style="padding:16px 0;">
    <p>Try dismissing using the X button, pressing Escape, or clicking outside the card. The close reason will appear below.</p>
  </div>
</sp-splash>

<script>
  document.getElementById('showBtn3').addEventListener('click', function() {
    document.getElementById('splash3').show();
  });
  document.getElementById('splash3').addEventListener('splashClose', function(e) {
    document.getElementById('closeLog').textContent = 'Closed via: ' + e.detail.reason;
  });
<\/script>
`} />
